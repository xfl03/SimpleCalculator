# SimpleCalculator
Simple Calculator in Kotlin
## Build
````
.\gradlew clean build
````

## Run
````
java -jar .\build\libs\SimpleCalculator-1.1-SNAPSHOT.jar -m
````
### Argument
`-d` `--debug` DebugMode  
`-f` `--fix` Auto Fix Right Bracket in the end  
`-m` `--mixed` Use Mixed Fraction  
`-i` `--int` Integer Only
